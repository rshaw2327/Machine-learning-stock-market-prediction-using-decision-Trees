# рџљЂ Project Overview

This project demonstrates how **machine learning can be applied to financial markets** by predicting whether a stock price is likely to **move up or down** in the next time step.

The system:

1. Downloads historical stock data using `yfinance`
2. Generates financial features such as returns and moving averages
3. Trains a **Decision Tree classifier**
4. Predicts the next price direction
5. Displays results on a **web dashboard built with Flask**

The dashboard also visualizes recent price data using **interactive candlestick charts**.

---

# рџ§  Machine Learning Model

The prediction model is a **Decision Tree Classifier**.

### Features Used

* Daily Returns
* 10-day Moving Average
* 50-day Moving Average

### Target Variable

Binary classification:

* **1 в†’ Price increases**
* **0 в†’ Price decreases**

The model is trained using an **80/20 train-test split**.

Metrics displayed on the dashboard include:

* **Prediction Direction**
* **Prediction Confidence**
* **Model Accuracy**
